Lua Script
In-game ranking commands for Roblox Studio
Configure Your Script
1
Download the script above
Click "Download Lua Script" — it will download a .lua file pre-configured with your workspace ID and group ID.
2
Open Roblox Studio
Go to your game → ServerScriptService → right click → Insert Object → Script.
3
Paste the script
Open the downloaded file, copy all the contents, and paste into the new Script in ServerScriptService.
4
Add the notification LocalScript
In StarterPlayer → StarterPlayerScripts, add a LocalScript and paste this:

local RS = game:GetService("ReplicatedStorage")
local SG = game:GetService("StarterGui")
RS:WaitForChild("RankNotification").OnClientEvent:Connect(function(t,m)
  SG:SetCore("SendNotification",{Title=t,Text=m,Duration=6})
end)
5
Enable HTTP Requests
Game Settings → Security → turn on "Allow HTTP Requests".
6
Use the commands!
In game chat type: !promote Username · !demote Username · !setrank Username 222
