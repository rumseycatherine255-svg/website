Features

Everything your group needs

Built specifically for Roblox café and community groups. No setup headaches.

⚡
One-click ranking
Promote, demote, or set any rank directly from the dashboard or in-game chat commands.
👥
Multi-workspace
Manage multiple groups from one account. Perfect for group owners running several communities.
📋
Full audit log
Every rank change is logged with who did it, when, and what the new rank is.
🛡️
Protected ranks
Set a protected rank threshold — co-owners and above can never be accidentally ranked down.
🎮
In-game commands
Use !promote, !demote, and !setrank directly in Roblox chat. Ranks update in real time.
🔐
Staff accounts
Add moderators with their own login. Admins see everything; moderators can only rank.
